# algorithm-practice

本仓库用来记录我的数据结构与算法练习代码

主要内容包括：
- LeetCode 刷题
- Codeforces 练习
- 蓝桥杯等常见算法比赛练习

---

### 仓库架构
algorithm-practice

├── code

├── problem

└── index

### code
存放所有题目的实现代码

题目按照该题的算法类型进行分类

每个文件对应一道具体算法题的代码实现

### problem
存放题目的题目信息和题解思路

题目按照该题的算法类型进行分类，与 code 分类一致

### index
存放题目索引和网址、

题目按照该题所属网站分类

---

## 目标

建立一个能长期维护的算法练习仓库，用以：
- 记录算法学习过程
- 提高编程的思维和能力
- 方便日后复习与总结
